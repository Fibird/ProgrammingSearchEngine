# 使用方法

1. 安装python 3.4+环境

2. 安装依赖包：

```
pip install lxml
pip install jieba
pip install Flask
pip install requests
pip install sklearn
```

5. 进入web文件夹，运行main.py文件
6. 打开浏览器，访问http://127.0.0.1:5000/ 输入关键词开始测试

如果想抓取最新新闻数据并构建索引，那么进入code文件夹，然后运行setup.py，再按上面的方法测试。
